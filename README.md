# Resume Builder

A dynamic Resume Builder created with HTML, CSS, and JavaScript. This application allows users to enter their information through input fields and automatically generates a professional resume layout in real time.

##  Features

- Personal information section
- Education details
- Skills and certifications
- Projects and experience
- Dynamic content generation using JavaScript
- Easy to customize

##  Tech Stack

- HTML5
- CSS3
- JavaScript
